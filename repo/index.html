Ludvig Eriksson's beta repository

This is where betas of updates and upcoming tweaks might turn up. To download and install them, add this repo to your sources.

Step 1

Under your Cydia sources tab, tap 'Edit' in the top right corner.

Step 2

Tap 'Add' in the top left corner.

Step 3

Enter the repo address and tap 'Add Source'.

Open sources tab in Cydia

Note

Cydia might encounter an error when adding the source if the repo currently is empty.
