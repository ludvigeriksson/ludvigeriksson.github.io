LETimeIntervalPicker

The UIDatePicker for time intervals

It's a mystery why a UIDatePicker can handle hours and minutes but not seconds. But there is a simple solution: LETimeIntervalPicker.

LETimeIntervalPicker sends a UIControlEvents.ValueChanged event when the user picks a time interval. Just listen for this event (also connectable via storyboards).

To get the time interval, simply use the property 'timeInterval'. If you want the time interval as hours, minutes and seconds, use the property 'timeIntervalAsHoursMinutesSeconds' instead. This is a tuple of (Int, Int, Int).

To set the time interval, also use the property 'timeInterval'. If you want to set the picker's time animated, use the function 'setTimeIntervalAnimated(timeInterval)' instead.

Download

LETimeIntervalPicker is available through CocoaPods. Just add the following line to your podfile:

pod 'LETimeIntervalPicker'

If you rather do it the old fashioned way, download the source code from Github, the link is below.
